What is the difference between Margin, Border, and Padding


Margin


The area that surrounds the elements position once you create a box element and give it a "margin: 10px;" you would create an atmosphere of 10px, seperating that box element from others around it


You would use the css code: margin-top, margin-right, margin-bottom, and margin-left to set the desired margin conditions


Border


The border is an edge or perimeter of a given element Borders can be given different visual choices in the forms of width and color to acheive the creators choice


Border codes are like: border-color, border-width, and border-style


Padding


Padding is similar to creating the margin the difference is that margin is the space between elements and padding creates the space between the border and the contents within the element


the padding codes are: padding-top, padding-bottom, padding-left, padding-right
